**Title:**
Keycloak Authentication Integration

**Objective:**
Integrate a centralized authentication system with Keycloak to ensure only authorized users can access the application and its data.

**Expected Result:**
All application access is protected, redirecting unauthenticated users to a standard login page. The UI is updated to display the user's name and email, along with a logout option. API communication is secured by including authentication credentials in every request.
