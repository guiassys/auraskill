**Title:**
Implement Internationalization (i18n) for English and Portuguese

**Objective:**
Implement full i18n support to display UI content in either English or Brazilian Portuguese based on user's regional settings.

**Expected Result:**
The application automatically detects the user's language and redirects to the appropriate localized route (e.g., `/en` or `/pt`). All static text is translated, and the URL reflects the current language. English is the default language.
